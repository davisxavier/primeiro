Site de Estudos - Portfólio Front-End
Este repositório contém um projeto de estudos focado no desenvolvimento de interfaces web utilizando HTML5, CSS3 e JavaScript. O objetivo é aplicar conceitos fundamentais de estruturação de páginas, estilização responsiva e manipulação de eventos e dados com JS.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica de páginas.

CSS3: Estilização, layout flexível e responsividade (Media Queries).

JavaScript: Lógica de interface, manipulação de DOM e sistema de busca dinâmica.

📁 Estrutura do Projeto
O projeto é composto pelas seguintes páginas e funcionalidades:

1. Página Principal (index.html)
Página de entrada que consolida o acesso aos mini-projetos e links de contato.

2. Mostrador de Horário (mostrador.html)
Exercício dinâmico que exibe uma saudação e uma imagem que se altera automaticamente de acordo com o horário do dia (manhã, tarde, noite ou madrugada).

Script: index.js

Estilo: estilo1.css

3. Mapa do Site (mapa.html)
Uma página dedicada a documentar a hierarquia de arquivos do projeto, servindo como um guia de navegação e organização técnica.

4. Sistema de Busca de Conteúdo (app.js & dados.js)
Funcionalidade inspirada na "Imersão Alura", onde o usuário pode realizar buscas em uma base de dados JSON interna. O script filtra títulos e descrições, renderizando os resultados dinamicamente na tela.

5. Outros Recursos
Fotos.html: Demonstração de uma loja de consignados com efeitos CSS em imagens.

Responsividade: O site foi adaptado para funcionar corretamente em desktops, tablets e smartphones.
